A clean, modern, and minimal developer portfolio built to showcase my projects, skills, and interests. Designed with a focus on smooth user experience, responsive layouts, and subtle interactive animations.

## 🌐 Live Demo

**[View the live site here](https://ashwinbhatk.vercel.app)**

## ✨ Features

- **Modern & Minimalist Design**: Clean typography (Inter font) with soft shadows and balanced spacing.
- **Dark/Light Mode**: Fully supported theme toggling using Tailwind CSS.
- **Smooth Animations**: Scroll-triggered fade-ins and interactive hover effects powered by Framer Motion.
- **Responsive Layout**: Optimized for desktop, tablet, and mobile viewing.
- **Component-Driven**: Built with a modular React architecture for easy maintenance and scalability.

## 🛠️ Tech Stack

- **Framework**: [React](https://react.dev/) (via [Vite](https://vitejs.dev/))
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Deployment**: [Vercel](https://vercel.com/)

## 📂 Project Structure

```text
src/
├── components/
│   ├── About.jsx       # Personal information & skills
│   ├── Contact.jsx     # Social links & contact info
│   ├── Footer.jsx      # Footer section
│   ├── Hero.jsx        # Landing section
│   ├── Interests.jsx   # Learning progress & hobbies
│   ├── Navbar.jsx      # Navigation & theme toggle
│   └── Projects.jsx    # Project showcase
├── App.jsx             # Main layout combining all sections
├── index.css           # Tailwind configurations & custom CSS
└── main.jsx            # Application entry point
```
